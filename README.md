## TrollGUI plugin
A simple troll plugin with a built-in GUI, which will make you able to troll a player with a simple click.
Features the following trolls:
- Creeper Troll
- Spider Troll

### Contribution
Feel free to make a pull request, with the following format:
`<feature/fix>/<name>`
